# Notes

## Angular CLI commands:
#### To create a new project -> First go to that folder where you want to create the project, then run
- ng new project_name

#### To start running the application -> Run this in terminal
- ng serve

#### To create a new component -> Run this
- ng generate component component_name
- Short form -> ng g c component_name

#### To generate servie
- ng generate service folderName/serviceName

#### To generate pipe
- ng generate pipe folderName/pipeName
